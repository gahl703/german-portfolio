Real Estate Management Project

James Madison University * Spring 2025 * Group Project (2-4 members)

Project Overview

  • Duration: 4 Months
  • Team Size: 2 Developers (expanded to 4 in final phase)
  • Technology: Java

    • Developed a Real Estate Management System for a simulated client, progressing through multiple development phases including prototyping, data modeling, object-oriented programming, and GUI planning.

Part One: Prototyping and Planning

  • Identified User Requirements
  • Created Wireframes on Gliffy to map out the early system prototype
  • Designed a Java Amortization Table for the monthly information

Part Two: Information Storage Basics

  • Assisted in developing a functional prototype of the entire system by utilizing arrays, loops, and variables
  • Researched and implemented File I/O to enable secure data saving and loading from a text file
  • Troubleshot implementation challenges through iterative testing and instructor feedback

Part Three: Fine Tuning the Data Model

  • Refined the system by implementing Classes and Inheritance, which ensured reusability, maintainability, and flexibility within the code
  • Formulated a class diagram as a blueprint for the project
  • Utilized Draw.io to create an Entity Relationship Diagram (ERD) for future database
  • Consulted with the client to ensure the system met their requirements

Part Four: Final Deliverable

  • Collaborated with another team to create a finalized system
  • Documented project requirements, which included the combined Entity Relationship Diagram (ERD) and class diagram
  • Updated wireframes for the new GUI system.
  • Integrated concepts from both teams to develop a fully functional real estate system.
  • Delivered a professional presentation of the project to the client.

Key Takeaway

This project enhanced my understanding of Java by enabling me to develop a fully functional Real Estate Management system for a client. I collaborated with a partner and later with another team, using different ideas to construct this system. Communication, time management, and teamwork tested me in various ways to ensure I met and completed the requirements before the deadline.

Tools

  • Gliffy and Draw.io
  • Apache NetBeam
  • Word Document
  • Use Case Modeling
