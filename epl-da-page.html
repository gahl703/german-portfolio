English Premier League Data Analysis

James Madison University * Fall 2025 * Group Project (3 members)

Problem Statement

The EPL Data Analysis project focused on transforming raw English Premier League player data into meaningful visualizations and actionable insights for stakeholders. Using Python libraries such as Pandas, Matplotlib, and NumPy, the project explored player trends and performance-related questions through data-driven analysis.

My Role:

Data Analyst

What I did:

  • Cleaned and evaluated raw datasets using Excel and Python to identify inconsistencies, null values, and potential bias.
  • Used Jupyter Notebook with Pandas, Matplotlib, and NumPy to analyze and visualize player performance trends.
  • Reassessed analytical assumptions after identifying skewed visualizations that did not accurately represent the dataset.
  • Collaborated with teammates to refine research questions better aligned with the available data.
  • Developed a Tableau dashboard after presentation feedback to create a more interactive and user-friendly reporting experience.

Early Challenge

One of the project’s biggest challenges involved recognizing that the original analytical question introduced bias into the visualization process. Rather than forcing misleading conclusions, the team reevaluated the dataset and adjusted the analytical direction to better reflect meaningful player trends.

The project demonstrated the importance of validating assumptions early, adapting analysis to the structure of the data, and communicating findings through clear visual storytelling. The resulting visualizations and Tableau dashboard improved the clarity and accessibility of the project’s insights.
